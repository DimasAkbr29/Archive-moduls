Gabut Changelog

Version 1

    This module automatically locks the CPU and sets the governor to userspace in game mode, maximizing CPU & GPU performance while disabling thermal throttling to ensure stability. It also turns off logging and debugging to reduce system load, helping to reduce FPS drops during gaming.

  • Added touch improvements for gaming
  • Enhanced CPU & GPU performance stability
  • Disabled logging & debugging to reduce FPS drops
  • Optimized system load management in game mode
