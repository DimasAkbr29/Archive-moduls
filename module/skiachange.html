SkiaGl Changelog

Optimized system performance for a more responsive UI and gaming experience by disabling blocking of untrusted touches, enhancing touch boost for faster response, and forcing GPU for touch rendering to improve visual performance. Additional features include smoother transitions, reduced touch latency, improved frame rendering, enhanced gaming performance by prioritizing touch response and reducing lag, efficient resource utilization to keep essential apps in memory and reduce RAM usage, dynamic CPU core allocation for better multitasking, enabling game mode for lower touch latency, and faster app opening and closing for a smoother experience.
